# Mesh Node T114

<div style={{ textAlign: 'center' }}>
  ![](img/1.jpg)
</div>

Mesh Node T096 is a low-power development board integrating Bluetooth, LoRa, and GNSS. Based on the nRF52840 MCU, it features an SX1262 LoRa transceiver with PA (up to 28±1 dBm) and a UC6580 GNSS module for positioning. It adopts LDS antennas for compact and efficient RF performance, and is optimized for battery-powered applications. Fully compatible with Arduino and open-source mesh systems such as Meshtastic and MeshCore, it provides a reliable platform for mesh networking and tracking applications.

## Product characteristicss

- Integrated Wi-Fi/Bluetooth, LoRa, and GNSS connectivity.
- Ultra-low power design for long battery life.
- Up to 28±1 dBm LoRa output power with LNA bypass support.
- Supports GPS, GLONASS, BDS, Galileo, NAVIC, and QZSS.
- Compatible with Arduino, Meshtastic, and MeshCore ecosystems.
