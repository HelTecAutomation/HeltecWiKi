HTCC-AB01 (V2) is based on ASR6052, the chip is already integrated with the PSoC® 4000 series MCU (ARM® Cortex® M0+ Core) and SX1262. We have done a lot of migration and development, made it perfectly support Arduino®, can run the LoRaWAN protocol stably, and can easily connect lithium batteries and solar panels. HTCC-AB01 (V2) is a Dev-Board. Which is an upgraded version of HTCC-AB01, and can pin-to-pin placement with the previous version.

## Product characteristics
- Arduino-compatible, CE & FCC certified
- Based on ASR605x (integrated PSoC® 4000 MCU + SX1262), supporting LoRaWAN 1.0.2
- Ultra-low power design, only 3.5µA in deep sleep
- Power management: battery interface, protection, auto switching, solar charging
- Communication & interfaces: optimized RF, long range, protected USB, built-in CP2102
