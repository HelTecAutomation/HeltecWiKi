HT-CT62 is a compact, low-power LoRa/LoRaWAN module with ESP32-C3FN4 + SX1262, integrating Wi-Fi and LoRa for IoT applications like smart cities, farms, and homes.

  </a>

## Product characteristicss

- ESP32-C3FN4 (RISC-V, 160 MHz) microprocessor
- Wi-Fi/BLE connectivity with IPEX 1.0 socket
- LoRaWAN 1.0.2 supported, Arduino compatible
- Ultra-low power design, 10 µA deep sleep
- 1.27 mm stamp-hole SMT, optimized RF matching for long range
