# Wireless Tracker V2

### V2.3.1

- 2026-3-11 public sale

- RF & Wireless Performance:
  - LoRa TX power upgraded from 21 ±1 dBm to 28 ±1 dBm.
  - Upgraded FEM and added RF filter to improve efficiency and EMC performance.
  - Full ground shielding and unified ground plane to enhance RF stability and anti-interference performance.

- GNSS:
  - Upgraded active antenna with enhanced LNA for stronger reception and higher positioning accuracy.

- Power:
  - Optimized power layout for improved system stability.
  - Added solar panel interface for extended outdoor operation.

- Layout Optimization:
  - Optimized LoRa GPIO routing to reduce signal interference.
  - Updated header footprint for better manufacturability.

- Mechanical Design:
  - Added dedicated plastic bracket for display protection.
