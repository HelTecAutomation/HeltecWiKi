Wireless Stick is a development board for Wi-Fi, Bluetooth and LoRa. Its functions are basically the same as “WiFi Lora 32”, except that the screen is reduced to 0.49 inch and a shielding shell is added.

{<div className={styles.btnContainer}>
  <a href="https://heltec.org/project/wireless-stick-v3/" className={styles.btnLink1}>
    Product Page
  </a>
</div>}

## Product characteristicss

- ESP32-S3FN8 MCU, support Wi-Fi, LoRa and BLE
- SX1262 LoRa node chip, compatible with Meshtastic
- Integrated CP2102 USB to serial port chip
- Onboard 0.49-inch 64*32 dot matrix OLED display
- With good RF circuit design and low-power design
